# Tpp_Project

---

基于Flask的restful风格编程实现的仿淘票票项目 beta1.0

### #环境

- OS:MacOS
- IDE:Pycharm 2018.3
- Python:3.6
- Flask:1.0.2
- Mysql:5.7.21

---
## Utils
- Flask_Restful-实现Restful风格的API编程
- Flask-Paginate-实现分页功能
- Flask-SQLAlchemy-实现数据库操作
- Flask_Cache-配合redis非关系型数据库实现分布式缓存机制
- Flask_Mail-实现发送邮件操作
- Flask_Migrate-拓展数据库表结构

---
## 运行环境

- 操作系统：Windows、Linux、Mac OS
